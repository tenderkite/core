# Kite

用 vue 的方式写后端，高扩展性

## 你可以得到什么

- [x] 选项式写法
- [x] typescript
- [x] 支持模块扩展
- [x] 缓存
- [ ] 接口提取
- [ ] 数据库保存
- [x] 支持中间件
- [ ] 管理
- [ ] 支持热更
- [x] 多线程
- [x] 多实体
- [ ] 全局事件
- [ ] 定时器

## 概念

- 服务-service：业务层的基本对象，通过 hash 算法，分布到不同的线程
- 组件-component：用于扩展 service
- 中间件-middleware：用于前置处理消息
